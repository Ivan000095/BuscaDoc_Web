# <img src="https://cdn-icons-png.flaticon.com/512/2966/2966327.png" width="28" align="top"> BuscaDoc (Web Backend & API)

Una plataforma integral diseñada para conectar a los usuarios con profesionales de la salud y farmacias. Este repositorio contiene el núcleo web (Backend y panel administrativo) construido en Laravel, el cual gestiona la base de datos, la lógica de negocio y provee una API robusta para la aplicación móvil.

## <img src="https://cdn-icons-png.flaticon.com/512/1184/1184082.png" width="24" align="top"> Características Principales

* <img src="https://cdn-icons-png.flaticon.com/512/3063/3063134.png" width="16" align="middle"> **Directorio Médico:** Búsqueda avanzada de doctores por especialidad, rango de precios y disponibilidad.
* <img src="https://cdn-icons-png.flaticon.com/512/4320/4320337.png" width="16" align="middle"> **Localizador de Farmacias:** Gestión y ubicación de farmacias.
* <img src="https://cdn-icons-png.flaticon.com/512/2693/2693507.png" width="16" align="middle"> **Sistema de Citas:** Motor de agendamiento para pacientes y panel de control para médicos.
* <img src="https://cdn-icons-png.flaticon.com/512/4712/4712035.png" width="16" align="middle"> **Asistente Virtual Inteligente:** Chatbot integrado potenciado por IA capaz de interpretar consultas médicas en lenguaje natural y ejecutar búsquedas precisas en la base de datos en tiempo real.
* <img src="https://cdn-icons-png.flaticon.com/512/1077/1077063.png" width="16" align="middle"> **Gestión de Perfiles:** Autenticación de usuarios, subida de fotografías de perfil y administración de roles.
